opções para instalação:

    --install           | -i    - Instalação do BDS (Script Obrigatorio)
    --update            | -u    - Atualização do BDS (Opção quando houver uma atualização do Minecraft Bedrock)
    --backup            | -b    - Ativar o Backup do Mapa apos para o Servidor BDS (Opcional)
    --start-on-system   | -s    - Incializar o BDS a sistema iniciar e Deixar o BDS em segundo Plano (Script Obrigatorio)
    --ip                | -P    - Mostrar seu IPs
    --apache2 | --page  | -a    - Instalar o Apache2 para adicionar o Servidor ao acessae pelo Navegadores tipo o Chrome, Firefox ...
    --files             | -f    - Ativa acesso dos arquivos do Servidor via SMB (SAMBA) e FTP (VSFTPD)
    --update-script     | -u-s  - Atualizar o Script (Atualizações do script para correção de Bugs)
    --remover           | -r    - Remoção do Script, não será desistalado o BDS e outros programas já instalado


--install ou -i: Apos a instalação podemos já utlizar o servidor mas não terá o script para inicialização do servidor;

--update ou -u: Com os lancamentos continuos do Minecraft Bedrock edition temos que manter atualizado o software do servidor BDS;

--backup ou -b: Para quem não quer perder seus backups após desligar o Servidor podemos ter um backup não é, backups continuos será mais tarde;

--start-on-system ou -s: Para quem vai instalar o BDS é bom ter um script que já inicie quando o servidor para ou precissar ser reiniciado;

--ip ou -P: Mostrar seu ips locais e seu im externo (IP externo por enquanto so IPv4);

--apache2,pages ou -a: -;

--files ou -f: Para quém quer acessar seus arquivos do BDS teremos um servidor smb e ftp pré-configurado pelo script;

--update-script ou -u-s: Atulizaçoes que mantemos desses script;

--remover ou -r: para quem não gostou basta remover a pasta atual e ficar feliz, e depois nos falar que merda deu no "https://github.com/Sirherobrine23/Minecraft-Bedrock-auto-install/issues/new";
